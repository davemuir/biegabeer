Add image watermark to images uploaded to media library

== Description ==

<h3>Features:</h3>
<ul>
<li>Watermark position</li>
<li>Min image size filter</li>
<li>Image type filter</li>
<li>Old images processing via batch</li>
</ul>

== Installation ==

1. Upload archive content to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
